What is the Document Object Model (DOM)?

The DOM is the bridge that lets scripts interact with and modify the HTML/CSS content of a webpage.

How Do You Select An Element On The DOM?

To select an element on the DOM, you can use JavaScript methods like:

document.getElementById('id') – Selects an element by its id.
document.getElementsByClassName('class') – Selects elements by their class name.
document.getElementsByTagName('tag') – Selects elements by their tag name.
document.querySelector('selector') – Selects the first element that matches a CSS selector.
document.querySelectorAll('selector') – Selects all elements that match a CSS selector.

what is event delegation in the context of the DOM ,and why is it useful?

Event delegation is a technique where you attach a single event listener to a parent element to handle events from its child elements.

Why Useful:
Efficient: Fewer event listeners, improving performance.
Dynamic Elements: Works for newly added child elements.
Simpler Code: Reduces multiple event listeners.

How Do You Manipulate an Element's attributes and style using the DOM?

Manipulating Attributes and Styles:
Attributes:

Get: element.getAttribute('attr')
Set: element.setAttribute('attr', 'value')
Remove: element.removeAttribute('attr')
Styles:
Set: element.style.property = 'value'
Get: const value = element.style.property
Remove: element.style.removeProperty('property')
